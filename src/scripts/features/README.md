# Features

Этот слой содержит переиспользуемые бизнес-функции проекта согласно архитектуре Feature-Sliced Design.

## Структура

```
features/
├── burger-menu-init.ts      # Инициализация бургер-меню
├── hero-slider.ts           # Слайдер героя на главной странице
├── how-we-do-slider.ts      # Слайдер "Как мы работаем"
├── useful-articles-slider.ts # Слайдер полезных статей
├── reviews-slider.ts        # Слайдер отзывов
└── stock-slider.ts          # Слайдер акций
```

## Принципы

- Каждая feature экспортирует функцию инициализации
- Features должны быть максимально автономными
- Конфигурация передается через параметры функций
- Все features проверяют наличие DOM элементов перед инициализацией
- Используют неймспейсинг для избежания конфликтов

## Пример использования

```typescript
import { initHeroSlider } from '@scripts/features/hero-slider';

// Инициализация с дефолтными параметрами
initHeroSlider();

// Инициализация с кастомным селектором
initHeroSlider('#custom-hero-slider');
```

## Зависимости

Все слайдеры используют библиотеку Swiper с соответствующими модулями:

- `Scrollbar` - для прокрутки
- `Pagination` - для пагинации
- `Navigation` - для навигации
